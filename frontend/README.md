# JobVista

## Overview

JobVista is a web application designed to connect students with relevant job opportunities. It provides a platform for students to browse and apply for jobs, and for administrators to post and manage job listings and applicants.

## Features

**Student Features:**

* Browse and search job listings.
* View detailed information about jobs.
* Apply for jobs directly through the platform.
* Track the status of their applications.
* Manage their profile information.

**Administrator Features:**

* Create and manage job postings.
* View a list of all job applications.
* Manage applicants for specific job roles.
* Export applicant data in CSV format.
